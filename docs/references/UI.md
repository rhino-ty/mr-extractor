## UX 퀄리티 기준

### 필수

- 모든 버튼/슬라이더에 hover 상태 애니메이션
- 로딩 중 스피너 또는 인디케이터 항상 표시
- 에러 발생 시 친절한 한국어 메시지 + 원인 힌트
- 툴팁은 hover 0.5초 후 나타나게
- 빈 상태는 일러스트 + 안내 문구로 처리
- 페이지 전환 애니메이션: fade 또는 slide (200ms)

### 폰트/간격

- 타이틀 20px / 섹션 13px bold / 본문 13px / 캡션 11px
- 카드 내부 padding 최소 16px
- 아이콘 + 텍스트 간격 6~8px
